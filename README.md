## Computer Science Keys

This repository is just a collection of introductory articles on computer science (originally written in Arabic).
It's called "keys" as a metaphor that it's your way to get the closed doors opened and start your journey in the field of CS.

## Generate PDF

Make sure that [typst](https://typst.app/) is installed on your machine.
Then execute the following command on any file you want to read:
```bash
typst compile <file-path>

# For Example
typst compile ar/0-preface.typ
```
